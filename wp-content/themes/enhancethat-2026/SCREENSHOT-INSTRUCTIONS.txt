SCREENSHOT INSTRUCTIONS
=======================

To complete the theme package, please create a screenshot.png file:

1. Open http://localhost:8080 in your browser
2. Ensure the page is fully loaded with all animations
3. Set your browser window to 1200 pixels wide
4. Take a screenshot of the hero section and top portion of the page
5. Crop/resize the screenshot to exactly 1200 x 900 pixels
6. Save as "screenshot.png" in this directory (wp-content/themes/enhancethat/)
7. Delete this SCREENSHOT-INSTRUCTIONS.txt file

The screenshot will appear in WordPress admin (Appearance > Themes) as the theme preview.

Recommended tools:
- Windows: Snipping Tool, Snip & Sketch, or browser DevTools
- Mac: Screenshot (Cmd+Shift+4)
- Online: Any image editor to resize to 1200x900px

After adding screenshot.png, recreate the ZIP package.
